# ansible-role-languages

Installs certain programming languages on Arch Linux or Ubuntu based systems via package management.

## Requirements

- Arch Linux or Ubunutu
- Sudo permissions

## Role Variables

No variables

## Dependencies

No dependencies

## Example Playbook

```
---
- name: Playbook
  hosts: localhost
  connection: local
  roles:
    - ansible-role-languages
```
