ansible-role-languages
=========

Installs certain programming languages on Arch Linux based systems via pacman.

Requirements
------------

- Arch Linux (pacman)
- Sudo permissions

Role Variables
--------------

No variables

Dependencies
------------

No dependencies

Example Playbook
----------------

```
---
- name: Playbook
  hosts: localhost
  connection: local
  roles:
    - ansible-role-languages
```
